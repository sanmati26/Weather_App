Full Stack Weather App

A full-stack weather application built using Django and a frontend interface.
It fetches real-time weather data using a third-party Weather API.

Features
- Search weather by city
- Real-time weather data
- Clean UI using Tailwind CSS
- Backend API using Django

Tech Stack
- Frontend: HTML, CSS (Tailwind), JavaScript
- Backend: Django
- API: OpenWeatherMap (or your API name)
- Database: SQLite

Setup Instructions
1. Clone the repository
2. Install dependencies
3. Add your API key in .env
4. Run Django server
